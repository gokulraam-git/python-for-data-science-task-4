# Real-World Data Project

## Retail Store Sales Analysis Using Python

This project focuses on analyzing a real-world retail sales dataset using Python libraries such as Pandas, Matplotlib, and Seaborn. The project aims to identify sales trends, profit patterns, and important business insights through data analysis and visualization.

---

## Project Objective

- Analyze retail sales data
- Identify top-selling and high-profit products
- Perform statistical and correlation analysis
- Visualize sales and profit trends
- Generate meaningful business insights

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- VS Code

---

## Dataset Features

The dataset contains:

- Product Name
- Category
- Sales
- Profit
- Quantity

---

## Key Analysis Performed

- Statistical Summary
- Correlation Analysis
- Total Sales Calculation
- Highest Profit Product Identification
- Data Visualization using Charts and Heatmaps

---

## Visualizations Included

- Product vs Sales Bar Chart
- Sales vs Profit Scatter Plot
- Correlation Heatmap

---

## Key Insights

- Electronics products generated the highest sales
- Higher sales products produced higher profits
- Fashion products showed higher quantity sales with lower profit margins

---

## Project Outcome

This project helped in understanding how data science techniques can be applied in real-world retail business scenarios to analyze sales performance and generate actionable insights.
